# Bookstore API:

## Schema: Book, Author, Category

### Queries:

    Get all books

    Get book by ID

    Filter books by category

### Mutations:

    Add a book

    Update book details

    Delete a book

### Practice Goals:

Use arguments (filter, limit, offset)

Nested resolvers (get author info when querying books)


## Technologies used:

### Langugage: 
     TypeScript

### Node js Libraries:
    - graphql
    - graphql-tag
    - @apollo/server
    - @graphql-tools/mock
    - @graphql-tools/schema
